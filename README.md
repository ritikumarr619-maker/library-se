# Library Book Management System (Agile + Git)

## Overview
This project implements a simple Library Book Management System using Python.
The system is developed using Agile methodology across three sprints, with
Git-based version control and unit testing.

## Features
- Add and store books
- Borrow and return books
- Generate library status report

## Technology Stack
- Python
- unittest
- Git (Agile sprint-based workflow)

## How to Run Tests
```bash
python -m unittest discover -s tests -p "test_*.py" -v
